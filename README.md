## 笔记结构 🏗️

1. **标题** 🏷️：简要描述笔记的主题。
2. **简介** 🖋️：对主题的简要介绍。
3. **主要内容**：
    - **概念解释** 🧠：详细解释相关概念。
    - **代码示例** 👨‍💻：提供实际的代码示例。
    - **注意事项** ⚠️：列出常见的错误和注意事项。
4. **总结** 📝：对主题的总结和回顾。
5. **参考资料** 📚：列出相关的学习资源和链接。
